A modern **reel-style food ordering web application** inspired by Zomato and Instagram Reels. The platform combines short food videos with seamless ordering, real-time updates, and a dedicated **Food Partner dashboard** for vendors.

---

## üöÄ Features

### üí≥ Payment Features

* Secure online payment integration using **Razorpay**
* Support for UPI, Cards, Net Banking, and Wallets
* Real-time payment verification and order confirmation

### üë§ User Features

* User authentication (Login / Logout)
* Scrollable **reel-style food feed** (short videos)
* Auto-play videos on scroll
* Like & Save food reels
* Smooth scroll-to-top on Home click
* Place food orders directly from reels
* View **My Orders** with order history
* Reorder from previous orders

### üçΩÔ∏è Food Partner Features

* Food Partner authentication
* Dedicated **Food Partner Dashboard**
* Create & manage food items (video-based)
* Receive orders in real time
* Update order status (Accept / Prepare / Complete)
* View incoming & past orders

### ‚ö° Real-Time Capabilities

* Real-time order updates using **Socket.IO**
* Instant order notifications for food partners
* Live UI updates without page refresh
